# Hilipedia

## 简介
这个仓库储存了一个静态网页，包含了游戏中可能出现的所有丘丘语文本，以及一个非官方的词典。
希望有了它，您的游玩能更加轻松愉快😄。

## 访问速度慢？
如果您的Github连接时好时坏，请访问该网页在[Gitee上的副本](https://chen_zhanming.gitee.io/hilipedia/)，这边的速度会快非常多。我会将Gitee仓库的内容与更新后的Github仓库同步，如果出现两边不一致的情况，敬请刷新浏览器对该页面的缓存或耐心等待。
